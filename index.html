Why don't we use RSA anymore

The main reason that RSA is no longer used as the main form of cybersecurity is becuase we have better encryption algoithms that use much smaller numbers for the example above you use RSA-3 or the number of digits in the primes used is 3. Today for the minimum security we use RSA-617 or prime numbers that are 617 digits long.

A secondary reason though would be that a quantum computer can be created that would so efficently run Shor's algorthim making the requirements for RSA so insane that no non-quantum computer would ever be able to encode and decode messages. If we keep using the RSA-n notation then to defend against a quantum computer you would need RSA-10,000+ to defend(for a bit) against what could one day be a small quantum computer.
